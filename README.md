# Retail Sales Intelligence Dashboard

## Dashboard Overview
This project transforms raw transactional data into a high-level executive dashboard. It provides a synchronized view of monthly revenue trends, regional performance, and category profitability.

## Key Technical Features
* **Performance Control:** Implemented a technical range constraint ($A1:M201$) to optimize spreadsheet calculation speed.
* **Data Integrity:** Reconciled all pivot tables to a verified Grand Total of **₹2,831,548.00**.
* **Analytical Framework:** Integrated "Analyst Notes" using the Performance-Takeaway-Action model to provide strategic recommendations.

## Tools Used
* Google Sheets
* Pivot Tables & Pivot Charts
* Data Validation & Conditional Logic
